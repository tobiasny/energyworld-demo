# Software Saver — Customer Interview

## 1. Company & Branding

- **Project name**: Software Saver
- **Branding**: Default design system (no custom branding)

## 2. Purpose

- **What it does**: Calculate the cost savings of substituting software developers with Claude Code-powered developers
- **Primary user**: Managers

## 3. Core Features

1. Input cost per developer per hour (NOK), number of developers, hours per month
2. Show potential cost savings based on research-backed productivity multipliers (~2x)

## 4. Data

- **Inputs**: Cost per developer/hour (NOK), number of developers, hours per month (default 160)
- **Output**: Savings breakdown (annual, monthly, team size reduction)
- **Currency**: NOK only

## 5. Pages & Navigation

- Single page: input form + results side by side

## 6. Scope & Priorities

- **MVP**: Calculator with inputs + savings breakdown
- **Out of scope**: Nothing explicitly excluded
