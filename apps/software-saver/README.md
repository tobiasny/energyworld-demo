# Software Saver — MVP

## Spec

A savings calculator for managers to see the cost impact of equipping developers with Claude Code, based on research-backed productivity multipliers (~2x).

## Features

- Input form: cost per developer/hour (NOK), number of developers (slider 1-100), hours per month
- Real-time savings calculation with annual and monthly breakdown
- Team size reduction estimate
- Dark mode, single-page layout

## Running Locally

```bash
npm install && npm run dev
```

## Deployment

```bash
npm run build
```

Deploy contents of `dist/`.
