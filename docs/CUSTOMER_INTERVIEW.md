# Customer Interview Questions

Questions to ask before building a new customer app.

## 1. Company & Branding

- What is your company/project name?
- Do you have a logo or brand colors you'd like used?

## 2. Purpose

- In one sentence, what should this app do?
- Who is the primary user (role, technical level)?

## 3. Core Features

- What are the 3-5 key things a user should be able to do?
- Which single feature is the most important?

## 4. Data

- What kind of data will the app display or manage (e.g. lists, dashboards, forms)?
- Can you describe the main data objects and their fields?

## 5. Pages & Navigation

- How many distinct views/pages do you need?
- Can you sketch or describe the main screen layout?

## 6. Scope & Priorities

- What is the absolute minimum for a useful first version?
- Anything explicitly out of scope?
